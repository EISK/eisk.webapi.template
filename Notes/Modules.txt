	-- Eisk.WebApi.GraphQL + Eisk.DataServices.CosmosDb.DocumentDatabase

Template Options
	- DataProvider
		-- EFCore (default) with EFCore In-memory database and SQL Server (to be also tested in SQL Azure)
		-- CosmosDb.DocumentDatabase
	- ApiType
		-- REST (rst) (default)
		-- GrapghQL (gql)
		-- gRPC (grpc)
	- SampleInclude
		-- Yes (default): creates required logical layers and configurations based on a single domain
		-- No: no domain specific sample is provided (i.e. Employee domain)
	- SampleModelType
		- Creates singe entity sample for Controller and Domain Class
		-- Entity (default)
		-- Editor: same editor model will be used for View and Create/Edit
		-- ViewAndEditor: separate model will be used for View and Create/Edit
	- SampleIncludeBusinessRule
		-- Yes (default)
		-- No
	- SampleIncludeBaseClass
		-- Yes (default): contains IsA relation type, where DomainService utility class will be used as parent class. No custom override method will be provided when SampleIncludeBusinessRule: No
		-- No: contains HasA relation type, where DomainService utility class will be used as property. CRUD methods will be avaible in the sample domain service class, even if  SampleIncludeBusinessRule: Yes
	- AuthType
		-- None (default)
		-- IdentityServer
		-- AzureB2C
	- ErrorHandlingFilter
		-- None
		-- Custom (default)
	- LogType
		-- None (default)
		-- Log4Net
		-- SeriLog
	- ApplicationInsightType
		- None (default)
		- AzureInsight
	- Include gitignore
		- Yes
		- No (default)

Utilities
	- CleanSample
		- Provide PowerShell script to clean included samples (without changing existing code)

Project: Eisk.Bazor
	- Dedicated repo and devops ecosystem 
	- Dedicated doc tab
	- Required different object model: Server, Client, Shared
	- Consider Web API + Entity Framework + SQL Server as backend
	- Consider different domain architecture (i.e. CQRS)

Project: Eisk.Samples
	- There will be another repo called Eisk.Samples, that will provide alll samples with different combinations of template options
	- Code samples from Bridge (special folder)
	- Additional samples (with different combinations of template options)
	- Blazor samples
